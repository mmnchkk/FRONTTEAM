HTML (Markup Language)

Для основной конструкции нам понадобится язык разметки HTML 5.

Ссылка на подробную информацию

Обучающие видео про основы HTML

Вставка текста

<h1></h1>...<h6></h6> — заголовок
<p></p> — основной текст
<a href=""></a> — ссылка (путь)
<ol></ol> — нумерованный список
<ul></ul> — маркированный список
<li></li> — пункт списка

Форматирование текста

<b></b> — простой жирный текст
<strong></strong> — важный жирный текст
<i></i> — текст курсивом
<s></s> — перечеркнутый текст
<br> — перенос на строку

Элементы

<div></div> — контейнер
<img src="" alt=""> — картинка (alt - что будет вместо img, если она не прогрузится)
<audio></audio> — звук
<video></video> — видео

Форма

<form></form> — создание формы
<button></button> — кнопка
<input> — поле ввода
<label> — памятка для поля ввода
<textarea></textarea> — создаёт большие поля ввода

Таблицы

<table></table> — размещение таблицы
<caption></caption> — название таблицы
<thead></thead> — шапка таблицы
<tbody></tbody> — тело таблицы

<td></td> — одна ячейка таблицы
<th></th> — обозначение заголовка таблицы
<tr> — новая строка
