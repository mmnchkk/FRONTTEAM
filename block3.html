Обучающие видео про основы CSS

Вставка текста

color — цвет текста
font-size — размер шрифта
font-family — шрифт
font-weight — толщина
text-align — выравнивание текста
line-height — интервал текста

Фон

background-color — цвет фона
background-image — фоновая картинка
background-repeat — повторение картинки фона
background-position — позиция фона

Форматирование

width — ширина (min/max)
height — высота (min/max)
margin — внешние отступы
padding — внутренние отступы
border — вид границы элемента
border-radius — скругление углов

Отображение

display — положение элементов
flex-direction — колонна/ряд
align-items — выравнивание по вертикали
justify-content — выравнивание по горизонтали

Адаптивность

@media (min-width: __ and max-width: __) {
ваши стили - более детальная верстка для конкретных размеров
}

@media (max-width: __) {
ваши стили - более универсальная верстка с процентыми соотношениями
}
